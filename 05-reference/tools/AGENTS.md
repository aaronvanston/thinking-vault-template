# Tools Glossary

Reference documentation for tools, products, and services.

## Purpose

When processing content, products/tools are often mentioned without explanation. This glossary captures what they are and what they do.

## File Format

```markdown
---
created: YYYY-MM-DD
type: tool
category: [ide | agent | infrastructure | library | service]
website: [url]
---

# Tool Name

## What It Is

[1-2 sentence description]

## What It Does

- [Key capability 1]
- [Key capability 2]
- [Key capability 3]

## Why It Matters

[Why this tool is relevant to your work]

## Links

- [Official docs](url)
```

Use `05-reference/templates/tool.md`.
