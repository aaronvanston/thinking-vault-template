# Daily Update Template

For posting daily/weekly status updates (Slack, email, standup, etc).

---

## Monday Format (New Thread)

Start a new thread/message at the beginning of the week:

---

**Monday Xth – Friday Xth**

Monday Xth
• [Category]: [Item]
• [Category]: [Item with detail]
• 1:1s: [Name]

---

**Include on Mondays:**
- Week date range header
- Day header
- Any known leave for the week

---

## Tue–Fri Format (Thread Reply)

Reply in the same thread:

---

[Day] Xth
• [Category]: [Item]
• 1:1s: [Name]

---

## Common Categories

| Category | When to use |
|----------|-------------|
| Product | Feature work, prioritization, customer requests |
| Hiring | Offers, new starters |
| 1:1s | List names only |
| Leadership | Strategy, planning |
| Team | Helping people, onboarding |

---

## Sources

1. **Calendar** – 1:1s, major meetings
2. **Known priorities** – Current projects
3. **Team context** – Leave, milestones

---

## Style Notes

- High-level, not exhaustive
- Keep it scannable
- Don't include every meeting
- Don't mention tomorrow's items today
