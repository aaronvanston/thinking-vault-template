# Daily Updates

Daily status updates (for Slack, email, standup, etc).

## Structure

```
daily-updates/
└── [year]/
    └── [month]/
        └── YYYY-MM-DD-update.md
```

## File Format

```yaml
---
created: YYYY-MM-DD
type: daily-update
status: draft
week: YYYY-MM-DD to YYYY-MM-DD
---

# Daily Update – [Day] [Date]

## Post

[Day] [Date]
• [Category]: [Item]
• [Category]: [Item]

---

## Notes

- Internal notes, context, things not for the post
```

## Template

Use `05-reference/templates/daily-update.md`.
